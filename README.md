# Real-Time-Chat-Application-
# 💬 Real-Time Chat Application

A real-time web-based chat application built using **Spring Boot**, **WebSockets (STOMP/SockJS)**, and **Thymeleaf**. The application enables instant messaging between multiple users with low-latency communication using WebSocket technology.

## 🚀 Features

- 💬 Real-time messaging
- 👥 Multi-user chat
- 🔄 WebSocket communication
- 📡 STOMP messaging protocol
- ⚡ Low-latency message delivery
- 🎨 Responsive user interface
- 🏗️ Spring Boot backend
- 🌐 Embedded Tomcat server

## 🛠️ Tech Stack

- Java 17+
- Spring Boot 3
- Spring WebSocket
- STOMP
- SockJS
- Thymeleaf
- Maven
- HTML5
- CSS3
- JavaScript

## ▶️ Getting Started

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/spring-boot-websocket-chat-app.git
```

### Navigate to the project

```bash
cd spring-boot-websocket-chat-app
```

### Run the application

```bash
./mvnw spring-boot:run
```

or on Windows

```cmd
mvnw.cmd spring-boot:run
```

### Open in Browser

```
http://localhost:8080
```
